  
  


# Tags 标签

  
  


### 定义 / Definition

  
  


在复杂的信息群中，对内容的关键属性设置标签，同一个内容可设置多个标签，不同的内容可共享同一个标签，以达到让搜寻者快速定位到所需要查找的某一类或某个内容。

  
  


### 规则／Rule



移动端标签文字必须全显示，标签内文字允许换行。

只读型样式设计避免有点击感。
